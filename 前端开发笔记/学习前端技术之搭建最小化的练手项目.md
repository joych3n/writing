# 学习前端技术之搭建最小化的练手项目

在学习前端框架或库时，我们经常需要上手练习，在这里介绍搭建一个最小化的练手项目，以学习`Lodash`为例：

1. 首先，创建并进入项目目录：

```bash
mkdir learning-lodash && cd learning-lodash
```

2. 初始化项目：

```bash
npm init -y
```

3. 配置 `package.json` 并运行 `npm install` 安装依赖：

```json
{
  "name": "learning-lodash",
  "version": "1.0.0",
  "type": "module", // 启用 ES 模块
  "main": "index.js",
  "scripts": {
    "start": "node --watch src/index.js"
  },
  "dependencies": {
    "lodash": "^4.17.21" // 添加你需要练习的库
  }
}
```

4. 创建示例代码文件 `src/index.js`：

```js
import _ from "lodash";

// 准备测试数据
const array = [1, 2, 2, 3, 3, 3, 4];

// 测试常用方法
console.log("原始数组:", array);
console.log("分块后:", _.chunk(array, 3)); // 将数组分成长度为3的子数组
console.log("去重后:", _.uniq(array)); // 数组去重
```

5. 运行项目实时查看结果：

```bash
npm run start
```

这样你就可以快速搭建一个开发环境，随时测试和学习库的各种功能了。上述步骤同样适用于学习其他 JavaScript 库，只需要替换相应的依赖包即可。
